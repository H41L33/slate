# Project Context

## Goal
To provide a lightweight, accessible static site generator that treats Gemini and Gopher as first-class citizens alongside HTML.

## Key Features
-   Multi-protocol support (HTML, Gemini, Gopher).
-   GitHub Flavored Markdown (GFM) support.
-   IPFS compatibility (relative links).
-   RSS feed generation.
-   Simple, file-system based routing.
-   Tipping support (ETH/SOL addresses).
